# Password Strength Analyzer

## Overview
The Password Strength Analyzer is a web-based application that checks the strength of a user's password in real time. It provides instant feedback based on password length, uppercase letters, lowercase letters, numbers, and special characters, helping users create stronger and more secure passwords.

## Features
- Real-time password strength analysis
- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers
- Detects special characters
- Displays password strength as Weak, Medium, or Strong
- Simple and responsive user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.
4. Enter a password to see its strength.
